题目链接

洛谷

UVa Online Judge

题目翻译

在共 $N$ 轮的网球锦标赛中，第一轮有 $2^N$ 名球员被随机分配。随机分配意味着每个球员被分配一个介于 $1$ 和 $2^N$ 之间的数字，这个分配确定了两个球员之间是否会进行比赛。

特别地，第一轮的球员被编号为 $1,\ 2,\ …,\ 2^{N - 1}$。

在第 $k$ 轮中，编号为 $2 \cdot k - 1$ 的球员与编号为 $2 \cdot k$ 的球员进行比赛。

每一轮中的获胜者将晋升至下一轮，而输者将被淘汰，所以，下一轮中的球员数量会是当前轮的 $\frac{1}{2}$。每一轮中球员将会被重新编号，直到那一轮仅剩下一位选手，该选手就是冠军。

现在你需要得出在第一轮时编号为 $i$ 的选手与该轮时编号为 $j$ 的选手在第几轮可能会进行比赛。

题目解析

本题实际上就是一个二分答案，对球员编号从最后一轮往前不断二分直到满足 $i \leq mid$ 且 $j \geq mid$。

在进行二分答案前，需先保证 $i < j$，不满足则将 $i$ 与 $j$ 的值互换。

代码

#include <bits/stdc++.h>
using namespace std;

int n, i, j, l, r, mid;

int main() {
	while (cin >> n >> i >> j) {
		if (i > j) {
			swap(i, j); // 不满足 i < j 则将值互换
		}
		l = 1;
		r = pow(2, n); // 2^N (数据量较小直接 pow，也可使用 1 << N)
		mid = r / 2;
		while (true) {
			if (i <= mid && j > mid) { // 满足 i<= mid 且 j > mid
				cout << n << endl;
				break;
			}
			if (j <= mid) {
				r = mid;
			} else {
				l = mid + 1;
			}
			mid = (l + r - 1) / 2;
			n--; // 轮数往前推
		}
	}
	return 0;
}
